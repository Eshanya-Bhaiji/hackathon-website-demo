# hackathon-website-demo
CyberSec Hackathon Website

Welcome to the CyberSec Hackathon official website! This website is designed to provide participants, mentors, and visitors with a seamless experience to explore event details, register, view timelines, and verify certificates.

🌐 Live Demo

 chic-zabaione-a7f0b8.netlify.app

🛠️ Technologies Used
HTML5 – Structure and content of the website
CSS3 – Styling, animations, and responsive design
JavaScript – Interactivity, timeline animations, certificate verification, and QR scanning

Optional libraries/plugins:

OCR/QR libraries (for certificate verification)
AOS.js / GSAP for scroll animations (if implemented)
📌 Features
Home Section – Eye-catching hero section with event highlights.
About Section – Informative cards about the hackathon, goals, and rules.
Timeline Section – Curved, glowing circular timeline showing hackathon stages from registration to results.
Sponsors Section – Showcase of event sponsors and partners.
Certificate Verification – Manual entry, image upload (OCR), and QR code scanning options.
Responsive Design – Fully functional on desktops, tablets, and mobile devices.
Background Animations – Subtle cyber-themed animations for a modern look
